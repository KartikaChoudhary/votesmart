# VoteSmart India 🗳️
### AI-Powered Election Education Platform

Built for #BuildwithAI #PromptWarsVirtual — Challenge 2: Election Process Education

## 🔗 Live Demo
[Click here to view the live app](https://plum-carlin-1.tiiny.site)

## 📌 What it does
VoteSmart helps Indian citizens understand the election process through:

- 📅 **Election Timeline** — animated step-by-step journey from MCC to government formation
- 🗳️ **How to Vote** — interactive guide for first-time voters with EVM explanation
- ⚖️ **Compare Parties** — AI compares parties on issues YOU care about (jobs, water, farmers) in bullet points with Hindi translation
- 📍 **My Constituency** — enter pincode to find your MP, MLA and their report card
- ✅ **Register & Myths** — voter registration steps + busting common election myths

## 🌐 Languages
English + Hindi (हिंदी) toggle on all content

## 🛠️ Built With
- HTML, CSS, JavaScript
- Google Gemini AI API
- Cloudflare Workers (secure API proxy)
